# Framer Motion Demo

Ein kleines React + Vite + TypeScript Projekt mit drei Framer-Motion-Beispielen:

- **FadeInHeader** – Fade-in-Animation beim Laden
- **HoverTiles** – Hover-/Tap-Animationen mit Spring-Physik
- **AnimatedList** – Ein-/Ausblenden von Listenelementen mit `AnimatePresence`

## Lokal starten

```bash
npm install
npm run dev
```

## Auf GitHub veröffentlichen

1. Neues Repository auf GitHub anlegen (ohne README/gitignore, das ist hier schon vorhanden).
2. In diesem Ordner:

```bash
git init
git add .
git commit -m "Initial commit: Framer Motion demo"
git branch -M main
git remote add origin https://github.com/<dein-username>/<repo-name>.git
git push -u origin main
```

## Build

```bash
npm run build
npm run preview
```
